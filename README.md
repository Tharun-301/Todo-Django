# 📖 Project Overview

The **Django Todo App** is a beginner-friendly project built using Django that demonstrates complete **CRUD (Create, Read, Update, Delete)** functionality.

The application allows users to:

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- 🔄 Restore completed tasks
- 🗑️ Delete tasks

The interface separates **Pending Tasks** and **Completed Tasks** for better task management.

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Tharun-301/Todo-Django.git
```

```bash
cd Todo-Django
```

---

## 2️⃣ Create Virtual Environment

### Windows

```bash
python -m venv env
```

```bash
env\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv env
```

```bash
source env/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Apply Migrations

```bash
python manage.py migrate
```

---

## 5️⃣ Run Development Server

```bash
python manage.py runserver
```

---

## 6️⃣ Open Application

```text
http://127.0.0.1:8000/
```

# 📸 Application Workflow

## ➕ Add Task

Create a new task from the input box.

Example:

```text
Learn Django
```

---

## ✏️ Edit Task

Modify an existing task.

Example:

```text
Learn Django REST Framework
```

---

## ✅ Mark as Done

Move a task to the **Completed Tasks** section.

---

## 🔄 Mark as Undone

Restore a completed task back to **Pending Tasks**.

---

## 🗑️ Delete Task

Permanently remove unwanted tasks.
